# ADR-006 - Utilisation de PostgreSQL comme base de données

- **Status** : Accepté
- **Date** : 2026-03-30

## Contexte

L'application manipule des données relationnelles :
- utilisateurs
- réservations
- places de parking
- historique

Elle nécessite également des transactions fiables.

## Décision

Nous utilisons PostgreSQL comme base de données principale.

## Alternatives considérées

- MongoDB
- MySQL
- SQLite

## Conséquences

- Forte cohérence des données
- Support des transactions
- Bonne modélisation relationnelle
- Nécessite une modélisation stricte
