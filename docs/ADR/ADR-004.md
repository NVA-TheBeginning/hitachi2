# ADR-004 - Utilisation de Fastify pour le backend

- **Statut** : Accepted
- **Date** : 2026-03-30

## Contexte

Le backend doit être performant, simple et rapide à mettre en place, avec un bon support TypeScript.

## Décision

Nous utilisons Fastify comme framework backend.

## Alternatives considérées

- Express.js
- NestJS
- Spring Boot

## Conséquences

- Performance élevée
- Simplicité de mise en œuvre
- Moins de structure imposée que NestJS
- Besoin de structurer manuellement les modules
