# ADR-003 - Utilisation de Next.js pour le frontend

- **Statut** : Accepted
- **Date** : 2026-03-30

## Contexte

L'application nécessite :
- une interface utilisateur simple
- plusieurs vues selon les rôles (employee, secretary, manager)
- un support PWA
- une rapidité de développement

## Décision

Nous utilisons Next.js avec TypeScript pour le frontend.

## Alternatives considérées

- Angular
- React (sans framework)
- Vue.js

## Conséquences

- Développement rapide
- Bonne intégration PWA
- Structure claire des pages
- Facilité pour gérer les rôles et dashboards
- Dépendance à l’écosystème Next.js
