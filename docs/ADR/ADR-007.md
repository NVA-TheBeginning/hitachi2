# ADR-007 - Utilisation de Prisma comme ORM

- **Statut** : Accepted
- **Date** : 2026-03-30

## Contexte

Le projet nécessite un accès base de données simple, typé et rapide à mettre en œuvre.

## Décision

Nous utilisons Prisma comme ORM.

## Alternatives considérées

- TypeORM
- Sequelize
- Requêtes SQL brutes

## Conséquences

- Productivité accrue
- Schéma lisible
- Migrations simplifiées
- Type safety
- Abstraction qui peut limiter certaines optimisations SQL avancées
