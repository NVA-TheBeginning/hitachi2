# ADR-001 - Utilisation d'une Progressive Web App (PWA)

- **Statut** : Accepté
- **Date** : 2026-03-30

## Contexte

L'application doit être utilisée par des employés non techniques sur différents supports (ordinateur et mobile).
Le besoin inclut un accès rapide, sans installation complexe, notamment pour des actions simples comme la réservation ou le check-in via QR code.
L'équipe a de l'éxpérience dans le développement de PWA.

## Décision

Nous choisissons de développer l'application sous forme de Progressive Web App (PWA).

## Alternatives considérées

- Application mobile native (iOS/Android)
- Application desktop
- Site web classique sans fonctionnalités PWA

## Conséquences

- Expérience proche d’une application mobile sans passer par un store
- Installation possible sur écran d’accueil
- Accès simplifié via navigateur
- Fonctionnalités hors ligne limitées (pas de logique métier hors ligne)
- Maintenance simplifiée (une seule base de code)
